### 대규모 목록 최적화

렌더링 목록은 React에 구성요소가 있기 때문에 꽤 일반적인 방법입니다,

큰 목록을 렌더링하는 것은 속도가 저하되고 많은 사용량이 발생할 수 있기 떄문에 권장되지 않습니다,

이러한 문제를 처리하는 가장 좋은 방법은, "단순히" 보이는것만 렌더랑 한다는 것입니다,

그 중 유명한 라이브러리는 `react-window` 와 `react-virtuoso`입니다
